# Smart Home AI

一个基于 AI 意图理解的智能家居助手项目。

本项目目标是让用户通过自然语言表达需求，例如“好热”“有点闷”，系统自动判断用户意图，并控制家电设备。当前阶段先使用虚拟设备模拟空调控制，后续接入 ESP32 + 红外模块控制真实老旧家电。

## 项目目标

- 支持自然语言输入
- 根据用户表达判断家居控制意图
- 控制虚拟设备
- 后续接入真实红外硬件
- 为老旧家电提供低成本智能化升级方案

## 当前功能

- 输入“好热”
- 后端判断需要开启空调
- 调用虚拟设备接口
- 页面显示 AI 判断结果
- 终端输出“空调已打开（模拟）”
