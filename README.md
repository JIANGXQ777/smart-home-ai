# Smart Home AI

基于 AI 意图理解的智能家居助手，让用户通过自然语言控制家电设备。

## 项目简介

本项目旨在为老旧家电提供低成本智能化升级方案。用户只需说"好热"、"好冷"等自然语言，系统自动判断意图，在用户确认后执行设备控制。

## 技术栈

- **前端**：HTML + CSS + JavaScript
- **后端**：Node.js + Express
- **AI**：V1 阶段使用规则模拟（后续接入大模型）
- **设备**：V1 阶段使用虚拟设备模拟

## 快速开始

### 1. 安装依赖

```bash
npm install
```

### 2. 启动后端

```bash
npm start
```

服务器启动后访问 http://localhost:5000

### 3. 测试前端

直接在浏览器中打开 `frontend/index.html`

## API 接口

| 接口 | 方法 | 说明 |
|------|------|------|
| `/api/state` | GET | 获取系统状态（环境信息 + 设备列表） |
| `/api/chat` | POST | AI 对话，返回建议动作 |
| `/api/execute` | POST | 确认执行设备动作 |

## V1 功能测试

| 输入 | AI 建议 |
|------|---------|
| 好热 / 太热 / 闷 | 打开卧室空调 |
| 好冷 / 太冷 | 关闭卧室空调 |
| 打开空调 | 打开卧室空调 |
| 关闭空调 | 关闭卧室空调 |
| 打开风扇 | 打开客厅风扇 |
| 开灯 | 打开客厅灯 |
| 你能控制什么 | 返回设备能力说明 |

## 版本规划

- **V1**：文字版 Demo，规则模拟 AI，虚拟设备
- **V2**：接入真实大模型 API
- **V3**：设备配对与红外学习
- **V4**：语音交互
- **V5**：真实硬件控制（ESP32 + 红外模块）
- **V6**：主动智能提醒
